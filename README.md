# DARLINGintheFRANXX
A DARLING in the FRANXX theme for BetterDiscord

## HOW TO DOWNLOAD

1. Click the DARLINGintheFRANXX.theme.css
2. Click RAW
3. Right click the page and save as
4. Put it in your BetterDiscord Theme folder
